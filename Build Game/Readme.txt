Aby Uruchomić system należy otworzyć plik Game.exe. Plik powinien uruchamiać się na maszynach z systemem operacyjnym Windows. W wypadku gdyby plik exe nie działał prosi się o ponowną kompilację projektu zgodnie z krokami podanymi w dokumentacji.

W folderze znajdują się pliki/foldery niezbędne do prawidłowego działania systemu. Są to:
* plik config.cfg
* folder dane
* folder resources

Oczekiwana struktura tych plików/folderów została podana w Dokumentacji.
